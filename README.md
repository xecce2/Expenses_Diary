# Expenses Diary 💰

## 📖 Description
Expenses Diary is a console-based Python application for tracking personal expenses.  
It allows users to record daily expenses, categorize them, and calculate total spending.

This project was created for educational purposes to practice working with file handling, JSON data, and object-oriented programming (OOP).

---

## ⚙️ Features
- Add new expenses
- View all recorded expenses
- Calculate total expenses by category
- Save and load data using JSON
- Simple and user-friendly console interface

---

## 🛠️ Technologies
- Python 3
- JSON (for data storage)
